A variável nome é do tipo string para armazenar o nome do usuário.

A variável idade é do tipo number para armazenar a idade do usuário.

A variável profissao é do tipo string para armazenar a profissão do usuário.

A variável objetivo é do tipo string para armazenar o objetivo do usuário no Floripa Mais Tec.

O prompt é utilizado para solicitar ao usuário as informações necessárias para preencher as variáveis.

A mensagem é exibida no console usando o método console.log().

// Declaração da variável
const idade_cliente = 21;

// Bloco if
if (idade_cliente >= 18) {
  console.log("cliente é maior de idade");
}


A variável idade_cliente é do tipo number para armazenar a idade do cliente.

O bloco if verifica se o valor da variável idade_cliente é maior ou igual a 18. Se for, o bloco é executado e a mensagem "cliente é maior de idade" é impressa no console.

Exemplo de execução:

cliente é maior de idade
